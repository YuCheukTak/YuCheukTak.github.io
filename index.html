Biography

IoT Smart Dumbbell Prototyping (Ongoing)

A self-directed development project aiming to create a digital motion tracker for physical training equipment. The design uses an ESP32-C3 microcontroller connected via an I2C bus interface to a GY-511 telemetry module (incorporating the LSM303DLHC accelerometer and magnetometer). Current tasks involve coding a basic complementary filtering setup to reduce motion artifacts and accurately register repetitive angular changes. Full data synthesis is slated for completion in June 2026.
